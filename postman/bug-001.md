# Bug 001 — Missing Fields Returns 500 Instead of 400

**Endpoint:** POST /booking  
**Severity:** Medium  
**Expected:** 400 Bad Request  
**Actual:** 500 Internal Server Error  

**Steps to Reproduce:**
1. Send POST request to /booking
2. Include only firstname field in the body
3. Observe response

**Impact:** API does not handle incomplete requests gracefully,
returning a server error instead of a validation error.
